# Mean-Project

### Overview 
This application is created for those who want to know that how to write well scalable code using [MeanStack], so this is the public repository for the students and other developers who want to see that how I write code, for using this project in your local system you need to come with following steps.

### Step 1
After clone completed go to server (Mean-Project > server) using `cd server` & run `npm i` or `npm install`, after installing all important modules which are used to making this application, simple run `npm start`, If you successfully followed these steps so your [Node] project will run.
